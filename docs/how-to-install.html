How to Install AnyDevLibrary

Follow these steps to get AnyDevLibrary set up in your projects.

1. Download the Library

First, download the latest model.jar file from our GitHub Releases page (or directly from the repository's root if you've uploaded it there).

2. For AnyLogic Models

  1. Open your AnyLogic model.
  2. In the "Projects" view, right-click on your model's name (the top-level item).
  3. Select "Properties".
  4. Go to the "Dependencies" section.
  5. Under "Jar files and class folders required to build the model", click "Add".
  6. Choose "Java Archive File (*.jar, *.zip)" and browse to the model.jar file you downloaded.
  7. Select "Import to model folder" for easier portability.
  8. Click "OK".

Your AnyLogic model can now access the library's classes and agent types.

3. For External Java Projects

If you're integrating the library into a standalone Java project (e.g., Maven or Gradle project):

Maven

Add the following dependency to your pom.xml:

<dependency>
<groupId>com.anydev.library</groupId>
<artifactId>anydev-library</artifactId>
<version>1.0.0</version> <!-- Replace with your actual version -->
